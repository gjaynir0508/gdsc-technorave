Pattern Generation

This Python code generates a recursive pattern:


    def generate_pattern(n):
        if n == 0:
            return ""
        elif n == 1:
            return "a"
        elif n == 2:
            return "b"
        else:
            return "a" + generate_pattern(n-1) + "c" + generate_pattern(n-1) + "b"

    pattern = generate_pattern(3)
    print(pattern)
